# CookieClicker 🍪
Cookie Clicker est un jeu incrémental où le joueur clique pour produire des cookies et débloque progressivement des améliorations automatiques.

Le but est d’optimiser sa production pour atteindre des quantités toujours plus massives grâce à une progression exponentielle. 🍪

## Langages 
HTML CSS JavaScript

## Lancer le serveur
```
npx http-server
```

### Before push
```
npm run lint
npm run prettier
```
